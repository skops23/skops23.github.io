# 🌐 Mi Página Web 

## - Actividad 1

## 📌 Descripción
Este proyecto es una página web básica desarrollada en HTML como parte de una actividad introductoria. 

En esta página presento mis primeros conocimientos sobre HTML y enlaces a páginas de mis compañeros.

---

## 🧠 ¿Qué aprendí?
- Estructura básica de HTML
- Uso de etiquetas como:
  - `<html>`, `<head>`, `<body>`
  - `<h1>`, `<p>`, `<a>`
- Cómo agregar enlaces a otras páginas
- Uso de GitHub Pages para publicar sitios web

---

## 🛠️ Tecnologías utilizadas
- HTML5
