HTML - Introduction

¿Que es HTML?
HTML es el lenguaje que se usa para crear la estructura de una página web. Es como el esqueleto de una página. Referencia:https://developer.mozilla.org/en-US/docs/Web/HTML

Head

El head contiene información que no se ve directamente, como el título o enlaces. Referencia: https://www.w3schools.com/html/html5_semantic_elements.asp

Body

El body contiene todo lo que sí ve el usuario. Referencia: W3Schools

Section

Sirve para dividir contenido en partes. Referencia: W3Schools

Article

Es contenido independiente, como una noticia. Referencia: W3Schools

Aside

Contenido extra o secundario. Referencia: W3Schools

Header

Parte superior de una sección. Referencia: W3Schools

Footer

Parte inferior de una sección. Referencia: W3Schools
